# Hola, soy Andrés Buitrago 👋
**Backend Engineer | Software Architecture & Data Optimization**

Me especializo en diseñar sistemas escalables de alta concurrencia y disponibilidad. Mi enfoque principal es transformar reglas de negocio complejas en arquitecturas limpias y eficientes, garantizando siempre la integridad y trazabilidad total de los datos.

### 🚀 En qué estoy trabajando ahora
* 👨‍💻 **Nourmesa:** Un sistema de gestión gastronómica (POS/KDS) de alto rendimiento construido bajo estándares de ingeniería robustos.
* ⚙️ **Arquitectura de Nourmesa:** Implementación con **Kotlin Multiplatform (KMP)** y backend centralizado, diseñado para sincronización de estados en tiempo real y flujos de datos optimizados para analítica predictiva.

### 🛠️ Tech Stack & Herramientas
* **Backend:** Java (Spring Boot), Go, Kotlin.
* **Bases de Datos:** Oracle DB (especialista en PL/SQL), PostgreSQL, MySQL.
* **Arquitectura:** Clean Architecture, SOLID, Patrones de Diseño (Factory, Strategy), APIs RESTful.
* **Infraestructura:** Docker, AWS, Análisis de Planes de Ejecución (Query Tuning).

### 🧠 Filosofía de Ingeniería
